══════════  INSTRUCTION SET (MEASURED, NOT DESIGNED)  ══════════

 READ(q)    winding number → topological, free, robust
 WRITE+1    2π phase ramp → exactly +1 quantum, else nil (quantized)
 ERASE      scalar bump (3-site) → one-way funnel toward q=0
 COMPARE    conditional branch on q value (READ + threshold)
 RESET      strong ERASE → guaranteed q=0 (funnel property)

   q=1 → q=2:  barrier ~1.0   →  need ~3.0 energy

══════════  FRAMEWORK BLUEPRINT  ══════════

 1. REGISTER TIER
    ring_t:  { double phases[N], double omega[N] }
    register_t: { ring_t *rings, int radix }
    · N=16 oscillators per ring, N/4-1 stable q-states
    · radix selects how many energy levels to admit

 2. INSTRUCTION TIER  
    typedef enum { READ, WRITE, ERASE, COMPARE, RESET,
                   KICK_AT, RAMP_AT } opcode_t;
    instruction_t: { opcode, site, amplitude, width }
    · WRITE = 2π ramp at specified site + width
    · ERASE = scalar bump at specified site + amplitude

 3. TRANSITION TIER
    T[Q_MAX][opcode_space] — precomputed, cached
    · measured via batched settle (μs per entry at N=16)
    · lens-editable: detune one site to reprogram T
    · proven composable: T² = physical two-step

 4. EXECUTION TIER
    clock = 1/K · (2π/N)²  ≈ 6.7/K  at N=16
    state: { q, microstate_phase[N], locked_flag }
    · SETTLE: integrate until phase-locked + q-stable
    · early-exit: flow-invariance gate (gaps ∈ (-π+m, π-m))

 5. RADIX CONTROL TIER
    BINARY:  q∈{0,1}     ERASE = NOT    WRITE+1 = SET
    TERNARY: q∈{0,1,2}   ERASE = DEC    WRITE+1 = INC
    N-ARY:   q∈{0..N}    each ring = log₂(N+1) bits
    MIXED:   per-ring radix (control=binary, data=n-ary)

══════════  KEY PROPERTIES  ══════════

 1. TOPOLOGICAL: q is a topological invariant — free read, robust
    to noise, survives small parameter changes

 2. ONE-WAY FUNNEL: scalar inputs always decrease |q|
    → natural reset mechanism, no external ground needed

 3. QUANTIZED INCREMENT: 2π ramp writes exactly +1, else nil
    → natural increment, no analog precision required

 4. MARKOVIAN: route-invariance proven (72/72 trials)
    clock-freedom proven (composition: 36/36)

 5. LOCALLY EDITABLE: detuning one oscillator changes T
    only near the defect, for robust states

 6. MARGINAL AS NONLOCALITY: |q| ≈ N/4 states carry
    edits globally → probabilistic/chaotic computing channel

 7. ENERGY-ORDERED: V(q) monotonic in |q|, barriers shrink
    with |q| → speed/stability/radix design triangle
